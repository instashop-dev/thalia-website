## 2. Style

| Field | Value |
|-------|-------|
| **Primary Style** | Glassmorphism + Flat Design |
| **Dark sections** | Deep navy hero (`hsl(218 44% 7%)`) with dot-grid texture + cyan radial glow |
| **Light sections** | White (`#ffffff`) and off-white (`hsl(220 20% 97%)`) alternating |
| **Card style** | Flat white with `border-border`, `rounded-2xl`, lift-on-hover (`translateY(-4px)`) |

| **Icon containers** | `w-12 h-12 rounded-xl`, `rgba(0,192,255,0.1)` bg, `rgba(0,192,255,0.18)` border |

### Style anti-patterns to avoid
- No skeuomorphic effects or heavy 3D ŌĆö keep it flat+glass hybrid
- No emoji as icons ŌĆö Lucide React only
- No random shadow values ŌĆö use `--card-shadow` and `--card-shadow-hover` tokens exclusively
- No mixed icon styles (stroke-width must be consistent = Lucide default `2`)

## 10. React / Tailwind Implementation Rules

- **No raw hex in Tailwind classes** ŌĆö use CSS custom property tokens via `hsl(var(--token))`
- **Inline styles only for dynamic/glassmorphic values** (rgba, blur, radial-gradient not in Tailwind)
- **`section-container` class** for all max-width wrappers ŌĆö never use `container` directly
- **Framer Motion `motion.div` + `{...inView(delay)}`** for all scroll-visible cards
- **`once: true` viewport** ŌĆö do not re-animate on scroll-up
- **Image `loading="lazy"`** on all below-fold images; `width`/`height` declared for CLS prevention
- **Platform logo `grayscale opacity-50`** default; `hover:grayscale-0 hover:opacity-90` on hover
- **AppCard uses `getAppLogo()`, `getPlatformColor()`, `getPlatformLabel()`** from `src/data/apps.ts`

- **No `rounded-full` on the Thalia logo** ŌĆö it must be square (`rounded-none`)
- **No trust counters inline in the hero** ŌĆö they live in the dedicated Stats section
- **No floating/looping animations on HeroVisual chips** ŌĆö entrance-only
- **No `mb-12` orphan margins** after CTAs unless followed by content
- **Email address as button label** ŌĆö use `Email Us ŌåÆ` text, not the full email string
- **No stock photos** in hero or values ŌĆö use the `HeroVisual` JSX component and icon cards
